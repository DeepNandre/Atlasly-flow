# Sub-Agent Prompt Pack

Use this folder to run one research agent per stage.

## How to use
1. Start each agent with `shared-context.md` content.
2. Append exactly one stage-specific prompt file.
3. Require output in the shared research format.
4. Store each agent's output in a dedicated file under `docs/agents/research/`.

## Files
- `shared-context.md`
- `stage-0-agent-prompt.md`
- `stage-0.5-agent-prompt.md`
- `stage-1a-agent-prompt.md`
- `stage-1b-agent-prompt.md`
- `stage-2-agent-prompt.md`
- `stage-3-agent-prompt.md`
